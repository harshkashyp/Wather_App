# 🌦️ Weather App

This is a simple and responsive Weather Forecast App built using **HTML**, **CSS**, and **JavaScript**, powered by the **OpenWeatherMap API**.

It fetches real-time weather data based on your location and displays current weather conditions, a 7-day forecast, humidity, wind speed, pressure, sunrise/sunset timings, and more — all in a clean and interactive interface.

## 🚀 Features

- 📍 Auto-detects user location using Geolocation API
- 🌡️ Shows current temperature, humidity, wind speed, and pressure
- 🌅 Displays sunrise and sunset time
- 📆 Real-time date and 12-hour clock with AM/PM
- 📊 7-day weather forecast with day/night temperature and icons
- 📱 Fully responsive design – works on all devices

---

## 🧰 Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla)  
- OpenWeatherMap API  
- Moment.js (for time formatting)
